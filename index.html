নতুন চ্যানেল যোগ করবেন যেভাবে

  1. রিপোতে থাকা channels.json ফাইলটি ওপেন করুন।
  2. একটি নতুন entry যোগ করুন — id, number, name, logo (ঐচ্ছিক) এবং url (m3u8 লিংক) দিন।
  3. Commit করে GitHub এ পুশ করুন — GitHub Pages কয়েক সেকেন্ডের মধ্যে আপডেট হয়ে যাবে।
  4. পেজ রিফ্রেশ করলেই নতুন চ্যানেলটি লিস্টে দেখা যাবে।
